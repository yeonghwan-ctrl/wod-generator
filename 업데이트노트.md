## 2026-06-04 — 계정 데이터 클라우드 저장(Firestore 연동)

- 비로그인은 기존 `localStorage` 유지. 첫 로그인 시 로컬 데이터를 클라우드로 이관,

- 변경 파일: `src/firebase.js`(db export), `src/store/useAthlete.js`(user 인자+동기화),
  `src/App.jsx`(`useAthlete(auth.user)`)
- 새 env 불필요(기존 Firebase 설정 재사용). 번들 크기 증가(Firestore SDK)
- ⚠️ 사용자 작업: Firebase 콘솔에서 **Firestore Database 생성**(서울 리전 권장) +
  보안 규칙(본인 문서만 접근) 등록 필요. 안 하면 저장이 동작하지 않음
- 빌드 통과, GitHub `main` 푸시
